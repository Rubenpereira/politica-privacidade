# Política de Privacidade

Os aplicativos desenvolvidos por **PU1XTB / Ruben Pereira** respeitam
a privacidade dos usuários.

## Coleta de dados
Os aplicativos **não coletam, não armazenam e não compartilham**
dados pessoais, sensíveis ou identificáveis dos usuários.

Não é necessário cadastro, login ou fornecimento de qualquer
informação pessoal para utilização dos aplicativos.

## Uso de dados externos
Alguns aplicativos podem acessar **conteúdos públicos**
hospedados em servidores externos, como o GitHub, exclusivamente
para fins de consulta de informações.

Esse acesso não envolve identificação do usuário nem envio de
dados do dispositivo.

## Anúncios
Os aplicativos **não exibem anúncios**.
